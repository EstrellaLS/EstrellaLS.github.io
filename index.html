Servidor Local Instalación

Servidor Local

Un servidor local es aquél servidor web almacenado en una red local al propio equipo de referencia. Nos ayudará a ejecutar elementos o programación web en nuestro pc, ya sean archivos php, html, javaScript,ect. También nos permitirá almacenar todos esos datos y/o elementos mediante MySQL(Un sistema de base de datos) sin peligro a perder ningún progreso. No requiere de la contratación de ningún dominio (Ya que es 127.0.0.1), por lo que el disco duro hace las funciones de hosting. Básicamente un servidor local es como un lugar de almacenamiento orientado a recopilar únicamente los datos de tu sistema, con la intención de utilizarlos y solicitar nueva información a los usuarios. Es completamente configurable en remoto y ampliable, ya que únicamente hay que modificar el hardware y/o software del equipo, actualizándolo periódicamente.

En el ámbito del desarrollo web resulta básico conocer el funcionamiento de un servidor local, ya que éste actuará de manera segura, sin perder ningún tipo de dato o información valiosa. Proporciona la confianza de poder realizar pruebas y trabajar sobre la página web sin peligro a perderla, y otra de sus grandes cualidades es que permite también que se realicen copias de seguridad sobre esta en cualquier momento.


Ventajas Desventajas
  • Ahorro de dinero: A largo plazo la inversión en un servidor local se amortigua. Realizando una inversión inicial gorda, pero beneficiosa.
  • Rápida conectividad: Aunque no se reciba una respuesta inmediata por el hecho de no encontrarse físicamente en el mismo lugar, la diferencia en la distancia se aprecia en pocas ocasiones.
  • Mayor seguridad: Un servidor local soluciona la posible situación de querer poder acceder físicamente a él. Teniendo acceso únicamente tú mediante algún algoritmo de cifrado.
  • Su mantenimiento: Es una tarea que requiere de mucho tiempo. Debido a que la gran mayoría de acciones no se pueden automatizar, es necesario tener un conocimiento exhaustivo del tema.
  • Dependencia: Si hubiera algún problema físico, como por ejemplo con el disco duro, producirá una caída temporal bastante extensa, dejando sin servicio a muchos clientes.
  • Gran consumo energético: Se realizará un gasto mayor de energía eléctrica, o en el servicio de Internet que se tenga.

LAMP

Un servidor local está definido con su propio nombre, ya que está compuesto por siglas, AMP(Apache, MySQL, PHP). Y dependiendo del sistema operativo en el que se utilice así recibirá el nombre; WAMP (en Windows AMP), LAMP (en Linux AMP) y MAMP (en Mac AMP).

L.A.M.P. es un acrónimo de Linux, el Sistema operativo que se utiliza. Apache, el Servidor web. MySQL, el sistema gestor de base de datos. Y PHP, el lenguaje de programación utilizado, PHPMyAdmin es una herramienta de software libre escrita en PHP que facilita el manejo de la administración y gestión de MySQL mediante un entorno gráfico vía web,conjunto de software de código libre. Se puede instalar de manera conjunta o independiente. Su objetivo es habilitar un servidor para el almacenamiento de las páginas y aplicaciones web dinámicas.

Web dinámica

AL utilizar la palabra "dinámica", nos referimos a que los elementos están en continuo cambio. En lugar de ser una página web únicamente informativa, la gran mayoría de webs que utilizan LAMP,WAMP o MAMP, son dinámicas. Ya que "interactúan" con el cliente o usuario, registrando información y gestionándola.

El servidor web y el cliente se comunican mediante los protocolos HTTP y HTTPS(Versión encriptada),que son una serie de reglas que permiten la transferencia de información. Los servidores locales son inprescindibles para que una página web sea dinámica, ya que se encarga de almacenar y organizar toda posible información que el usuario proporcione (de manera ordenada, mediante formularios o similares).

Aplicaciones y herramientas

Hay un amplio mercado de aplicaciones, herramientas y/o programas dedicados a la instalación y creación de los servidores locales, nombraré algunos de los más conocidos:

XAMPP

Es un programa multiplataforma disponible para Windows, Linux y MacOS, de código abiero y gratuito. Es probablemente el software más utilizado. Dispone de un servidor Apache y con los lenguajes PHP y Perl. +Info

WAMPServer

Es un programa que contiene Apache, MySQL y PHP para Windows, donde también encontramos PHPMyAdmin (Para gestionar las bases de datos. +Info

EasyPHP

EasyPHP es un WAMP orientado a desarrolladores PHP. La aplicación es muy sencilla y minimalista, ya que sólo muestra el control de los servidores y un registro de sucesos. Sin embargo, en la página local principal, ofrece un panel de opciones de cada servidor, cambiar credenciales, avisos de actualizaciones o incluso un intérprete para pruebas PHP. +Info

Laragon

Es un programa únicamente disponible para Windows, portable y dispone de un conjunto de herramientas almacenadas dentro de contenedores, ofreciendo así mayor garantía y seguridad. +Info
