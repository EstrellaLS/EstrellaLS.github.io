Servidor Local Instalación

Crear un servidor local con LAMP

Hay dos maneras de crearlo, mediante la terminal (de manera independiente), o haciendo uso de una de las aplicaciones. Yo he optado por hacerlo sobre una máquina virtual de Linux (Ubuntu) mediante su terminal. Muestro paso a paso cómo lo hice:

Los requisitos oficioales para montar un servidor LAMP son; 128MB de RAM, una CPU de 800MHz y 1GB de disco duro o ssd.

1.Para mayor facilidad, enetrar directamente como administrador, mediante el comando "$sudo -i"". E introducimos la contraseña corresponiente.

2.Actualizar repositorio, mediante el comando "$apt-get update"".

3.Instalar el servidor Apache, mediante el comando "$apt install apache2".

Y decir sí(S o Y)).

Se ha instalado la versión Apache 2.4.41 (Ubuntu).

4.Instalar MySQL, mediante el comando "$apt-get install mysql-server mysql-client".

Y decir sí(S o Y)).

Se ha instalado la versión 8.029-OubuntuO.20.04.3 para Linux en x86_64 ((Ubuntu)).

5.Para evitar problemas, modificamos el plugin del root mediante:

A)El comando: "SELECT user,authentication_string,plugin,host FROM mysql.user;". Para ver el tipo de autentificación de los usuarios.

B)Para modificar el plugin del root, x ejemplo con una contraseña de 0000. Utilizar el comando: "ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '0000';".

C)Volver a ver la autentificación de los plugins con el comando anterior: "SELECT user,authentication_string,plugin,host FROM mysql.user;".

D)Ejecutar el comando: "FLUSH PRIVILEGES;". Y después "quit", para salir.

6.Instalar el PHP y las librerías de conexión con apache y MySQL, mediante "$apt-get install php libapache2-mod-php php-mysql". Y decir que sí.

7.Instalar PHP MyAdmin, mediante "$ aptget instal phpmyadmin". Y configurar:

Pulsar la barra espaciadora para seleccionar el paquete.

Pulsar Enter para contiduar.

Pulsar Enter para contiduar.

Es recomendable escribir la misma contraseña (En mi caso "0000").

Confirmar contraseña y continuar.

8.Una vez instalado, abrir el navegador y buscar "localhost/phpmyadmin" en el buscador, seleccionar idioma y rellenar.

Esta es la vista de la página principal.

Y ya podemos hacer uso de manera sencilla mediante la interfaz gráfica de la base de datos.
